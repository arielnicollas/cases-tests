<!-- Parameters -->

robot -i tag -> Executa todos os testes relacionados a tag.

robot -e tag -> Executa todos os testes exceto o teste relacionado.

robot -d nome-da-pasta -> Executará o teste e irá organizar os resultados em
uma pasta.
<!-- É necessario passar o caminho da pasta onde se encontra o arquivo de teste para realizar a execução -->

robot -v  NOMEVARIAVEL:VALOR -> Subistitui o valor de uma variável para uma
execução específica.
